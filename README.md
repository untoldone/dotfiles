dotfiles
========

More of new machines setup script than a dotfiles project. Setup on Debian 10 or MacOSX with

    curl https://raw.githubusercontent.com/untoldone/dotfiles/master/bootstrap.sh | bash -s <SUDO_PASSWORD>
